<h1>Real-Time chat application</h1>

Chat application using laravel + livewire, laravel reverb.
<h2>Features</h2>
<ul>
    <li>Authentication</li>
    <li>User Profiles</li>
    <li>Friend System</li>
    <li>Groups(private/public)</li>
    <li>Private/Group chats</li>
    <li>Text prediction using ngram</li>
</ul>
